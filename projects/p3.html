Simvestify

Simvestify is a simple yet interactive stock simulation app that lets users buy, sell, and track stocks using real-time data—perfect for beginners who want to explore the world of investing without any financial risk.

Why This Project?

I’ve always been curious about the stock market—how prices move, how people make investment decisions, and how risky it can be without proper knowledge. So, I decided to create an app where users could practice investing in a completely safe environment. Simvestify lets users simulate buying and selling real-world stocks using virtual money, which makes it a great tool to learn and have fun without any actual loss. Plus, I wanted to build something that combines both programming and finance!

How It Works?

The app is built using Python and Streamlit, and it uses the yFinance library to fetch real-time stock market data. When users sign up, they get a starting balance, and they can use this virtual money to buy shares of their favorite companies. The app stores user info in a local SQLite database, so every time you log in, your portfolio is exactly where you left it. It also includes features like plotting recent stock trends and tracking owned shares, making it engaging and educational.

Takeaways

This was my first time building an app that involved user authentication, wokring with databases, and real-time API integration. It really helped me understand concepts like databases, session states, and object-oriented programming in a real-world context. I also explored design patterns like the Observer pattern to notify users of events, which added a nice touch to the user experience!

Future Scope

In the future, I plan to expand Simvestify with advanced features like stock watchlists, performance charts that includes analysis of the user portfolio, and maybe even predictive models for stock price trends or for suggested stocks to buy. I'm also considering better security for the databse by hashing passwords and several other features.
